Better Paths

A tool for easier Path creation and manipulation.

What is Better Paths?

Better Paths is a tool written in Kotlin designed to make it easier to create and manipulate Java's Path objects. It contains a lot of methods which mostly just act as extension functions to use Files's methods as if they were part of the Path class itself.

How does it work?

The tool is written in Kotlin and uses Java's Path class to create and manipulate paths.
It is designed for a more "Kotlin-esque" approach to path creation and manipulation. It also provides constructor-style methods for creating paths.

Why would I use it?

Better Paths provides Kotlin users a simpler and more intuitive way to create and manipulate paths. It improves readability and makes it easier to understand what is happening.

Example:


Java code: 


    public class Main {
        public static void main(String[] args) {
            // Get a path from a string
            Path path = Paths.get("/home/user/file.txt");
            
            // Read the contents of the file as a list of lines
            Stream<String> lines = Files.lines(path);
            List<String> linesList = lines.collect(Collectors.toList());
            
            // Compare with the readLines output
            assert Files.readAllLines(path).equals(linesList);
            
            // Output the lines to stdout
            lines.forEach(System.out::println);
        }
    }

Kotlin code (using Better Paths): 

    fun main() {
        // Get a path from a string
        val path = "/home/user/file.txt".toPath()

        // Read the contents of the file as a list of lines
        val lines = path.lines().toList()

        // Compare with the readLines output
        assert path.readLines().equals(lines)

        // Output the lines to stdout
        lines.forEach { println(it) }
    }

How can I get it?

Add a dependency to your build.gradle, build.gradle.kts, pom.xml, or other buildscript.

build.gradle: 


repositories {
    maven {
        url "https://maven.concern.i.ng"
        name "concernmaven"
    }
}

dependencies {
    implementation "sschr15.tools:better-paths:1.1.0"

    // If using in a Minecraft mod built with Fabric Loom, include it:
    include "sschr15.tools:better-paths:1.1.0"
}

build.gradle.kts: 


repositories {
    maven {
        url = uri("https://maven.concern.i.ng")
        name = "concernmaven"
    }
}

dependencies {
    implementation("sschr15.tools:better-paths:1.1.0")

    // If using in a Minecraft mod built with Fabric Loom, include it:
    include("sschr15.tools:better-paths:1.1.0")

pom.xml: 


<repositories>

    </repository>
